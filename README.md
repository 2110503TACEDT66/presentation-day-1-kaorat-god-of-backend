# Project starter with

```
npm init

npm i express dotenv mongoose cookie-parser
npm i -D nodemon
npm i jsonwebtoken bcryptjs
```

make sure you edit 

```json
  "scripts": {
    "dev": "nodemon server.js", // push this line
    "test": "echo \"Error: no test specified\" && exit 1",
    "start": "node server.js"
  },
```

# Run
```
npm run dev
```
